操作方法（ゲーム時）
  ・「←→」
      左、右に移動
  ・カーソルキー（Win32のみ）
      左、右に移動
  ・Escキー（Win32のみ）
      ゲームを終了

操作方法（ゲームオーバー時）

  ・「Retry」
      再度ゲームをプレイ
  ・「Exit」（ゲームオーバー時 && iOS以外）
      ゲームを終了
  ・Escキー（Win32のみ）
      ゲームを終了
  ・カーソル・スペース・Enter・Ctrl・右クリック・TAB（ゲームオーバー時 && Win32のみ）
      再度ゲームをプレイ

遊び方

  ・左右にラケットを移動します
  ・丸いボールが飛んでくるので下に行かないように跳ね返します
  ・丸いボールが飛んでくるので下に行くとゲームオーバー
  ・移動中に跳ね返せば点数1.2倍
  ・時々お邪魔イベントが起きる
  ・お邪魔イベント中は点数1.2倍(ワープ・クランクは1.5倍)

お邪魔イベント一覧

  ・Wind ←
    自機が常に左側に移動します
  ・Wind →
    自機が常に右側に移動します
  ・Gravity ↑
    ボールが常に上側に移動します
  ・Gravity ↓
    ボールが常に下側に移動します
  ・Skip
    ボールがスキップ移動します
  ・Sometimes Crank
    時々ボールがクランクのように曲がります（もっと酷い曲がり方になることもある）
  ・Sometimes Warp
    時々ボールがワープします
